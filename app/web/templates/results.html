Data Leakage in Prior Zero-Shot AMR Work
Prior work encoded gene features with 46-dim drug-class membership vectors — the exact labels being predicted — inflating zero-shot performance by 5.7× (0.775 → 0.137 ZS-within MRR after cleaning). Our clean evaluation uses ESM-2 protein embeddings with no drug-class information. The primary metric is ZS-all MRR (random = 1/46 ≈ 0.022), not the narrow 5-way ZS-within ranking used previously.

Ablation Study — Feature Necessity

Feature-MLP variants · Both ESM-2 and molecular fingerprints jointly necessary

Variant Gene Features Drug Features ZS-within MRR ZS-all MRR Δ vs Full
Feature-MLP (full) ESM-2 480d MolFP 3245d 0.137 ± 0.004 0.069 ± 0.006 —
FMLP: gene = 0 zeros MolFP 3245d 0.140 ± 0.003 0.020 ± 0.002 ↓ 3.5×
FMLP: drug = 0 ESM-2 480d zeros 0.010 ± 0.000 0.010 ± 0.000 ↓ 6.9×
FMLP: leaky 46d drug-class MolFP 3245d 0.159 ± 0.030 0.112 ± 0.000 ↑ 1.6× (leaky)
Interpretation: Setting gene features to zero collapses ZS-all MRR to random (0.020 ≈ 0.022), while zeroing drug features falls below random (0.010). Both feature modalities contribute independently; their combination provides genuine zero-shot transfer. The leaky variant confirms the 5.7× inflation is entirely attributable to drug-class label leakage in the gene feature vector.
